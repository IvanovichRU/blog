Instalación de Node.js para Windows

Organización de Archivos

Lunes, 20 de Marzo de 2023

Contexto

En mi clase de la materia organización de archivos, se nos asignó a mi y a mis compañeros, el día sábado 18 de Marzo un programa sencillo, se nos preguntó en qué lenguaje de programación desearíamos hacerlo, entre las opciones estaban C#, C++, Java, PHP y JavaScript (Node.js), yo recomendé utilizar JavaScript con Node.js dado que creo es un lenguaje bastante sencillo y fácil de adoptar por lo que llegamos a esta página un guía sencilla para instalar Node.js en Windows.

Instalación

Visita https://nodejs.org, da click en el botón del lado izquierdo que denota "XX.XX.X LTS". Es importante si es tu primera vez usando Node.js, asegurarse de instalar una versión LTS (Long Term Support) o Soporte a largo plazo en español. Una versión LTS a grandes razgos indica que será una versión con muy pocos bugs, muy pocos errores y nos dará menos problemas en general.

Esto descargará el instalador para Node.js, ejecutamos el programa que abrirá un diálogo de instalación, damos en siguiente o Next en todas las pantallas, a menos que quieras cambiar alguna configuración pero recomiendo no alterar ninguna configuración si es tu primera vez usando Node.js.

En cualquier consola de comandos escribe el comando node -v. Puedes buscar cmd y ejecutar la aplicación "Command Prompt" o powershell y ejecutar la aplicación "Windows Powershell" en el menú de inicio de Windows, ambas consolas funcionan sin problema, aunque personalmente recomiendo instalar Windows Terminal. Es una aplicación que permite abrir múltiples consolas entre las que tenga instalado en tu computadora como las dos anteriores mencionadas, aparte de tener la función de ser personalizable a un buen grado para configurarla a tu gusto.

Al ejecutar el comando mencionado, deberías ver algo similar a v18.15.0

🎉 ¡Felicidades, Node.js está correctamente instalado en tu computadora! 🎉
