Instalación de Node.js para Windows

Lunes, 20 de Marzo de 2023

Descargar Instalador

Visita https://nodejs.org, da click en el botón del lado izquierdo que denota "XX.XX.X LTS".

Esto descargará el instalador para la versión más reciente y estable de Node.js

Ejecutamos el instalador y aceptamos/damos siguiente en cada paso hasta finalizar la instalación.

En cualquier consola de comandos escribe el comando node -v. Puedes buscar cmd o powershell en el menú de inicio de Windows y ejecutar cualquiera de las dos, ambas consolas funcionan sin problema, aunque personalmente recomiendo instalar Windows Terminal.
